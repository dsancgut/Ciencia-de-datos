
# Trabajo póster ciencia de datos

Este repositorio contiene el código y los datos utilizados en el trabajo para realizar un modelo e ciencia de datos 

## Asignatura y grado:
Ciencia de Datos, grado en biomedicina (4º curso)

## Universidad:
Alfonso X el Sabio

## Descripción:

El proyecto tiene como objetivo desarrollar un modelo de regresión lineal predictivo que responda a esta pregunta de investigación [¿los pacientes con depresión más grave rEsponden mejor al tratamiento antidepresivo?]
